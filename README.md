# Aposta Inteligente

Aplicativo de simulação inteligente de apostas das Loterias Caixa.

## Objetivo
Permitir que o usuário simule apostas com base em análise estatística dos últimos 60 meses, respeitando rigorosamente as regras oficiais de cada modalidade.

## Modalidades previstas (fase inicial)
- Mega-Sena
- Lotofácil
- Quina
- Dupla Sena
- +Milionária

## Funcionamento
- O usuário escolhe a modalidade da loteria
- Define a quantidade de números permitida pela regra oficial do jogo
- O sistema gera até 5 jogos por simulação
- As combinações são baseadas em análise estatística de frequência histórica

## Modelo de cobrança
- Venda avulsa por simulação
- Valor: R$ 9,95

## Aviso legal
Este aplicativo não realiza apostas, não intermedia jogos e não garante resultados.
Não possui vínculo com a Caixa Econômica Federal.
Trata-se exclusivamente de uma ferramenta de simulação estatística e análise de dados públicos.
