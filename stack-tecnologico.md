# Stack Tecnológico — Aposta Inteligente

## Visão geral
O MVP será desenvolvido inicialmente como aplicação Web responsiva, com foco em rapidez, baixo custo e escalabilidade futura.

## Frontend
- Framework: Next.js
- Linguagem: TypeScript
- UI: Tailwind CSS
- Compatível com mobile (PWA futura)

## Backend
- API integrada ao Next.js (routes / server actions)
- Validação de regras por modalidade
- Execução do algoritmo de simulação
- Registro de compras e simulações

## Banco de Dados
- PostgreSQL
- Estrutura normalizada
- Histórico de simulações e pagamentos

## Pagamentos
- Mercado Pago
- Checkout transparente ou link de pagamento
- Liberação da simulação somente após confirmação

## Hospedagem
- Vercel (produção)
- GitHub como repositório oficial

## Segurança
- Sem promessa de ganho
- Logs de simulação
- Validação de parâmetros
- Limite de requisições por usuário
